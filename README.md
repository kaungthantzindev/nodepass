# nodepass

nodejs cli password generator

**Installation**

```bash
git clone https://github.com/kaungthantzindev/nodepass
cd nodepass
npm install
```

**run**

```bash
nodepass --help
```

**password length**

```bash
nodepass -l 30
```

**no number / no symbol**

```bash
nodepass -l 30 -nn -ns
```

**filesave**

```bash
nodepass -l 30 -nn -ns -s password.txt
```
